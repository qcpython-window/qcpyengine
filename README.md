Introduccion  
El módulo QCPython se esta  disenando  con el proposito de  proporcionar un marco para el desarrollo de aplicaciones y bibliotecas en el lenguaje Python . En ella se define e implementa la infraestructura del motor de Python (CPython), y proporciona una API para permitir a los desarrolladores registrar tipos y módulos  personalizados e integrar código Python con C++. 

Nota: El modulo QCPython esta creado para el framework Qt ya que su API esta basado casi en su totalidad con el sistema de objetos y metaobjetos de dicho framework.

Herramientas usadas
IDE: Visual Studio Community 2022 y extension Qt VS Tool 3.2.0
Compilador: msvc2022 
Version de Qt: Qt6.7.1_msvc2019
Version de Python: Python3.12.6
